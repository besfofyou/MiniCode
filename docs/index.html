MiniCode
terminal-first coding assistant

用更小体量，跑通完整 AI 编码工作流

MiniCode 是一个面向本地开发的轻量级终端编码助手。它保留了 model-tool 闭环、审批式文件修改、可扩展工具协议等关键能力，同时保持代码结构清晰，适合学习、改造与二次开发。

核心能力

围绕真实开发场景设计，而不是只做聊天窗口。

多步工具执行

在单轮中串联 list/read/edit/run 等操作，完成从理解到落地的完整闭环。

审批后写入

文件改动先展示 unified diff，再决定是否落盘，控制力与安全性并存。

终端体验优化

支持 transcript 滚动、输入历史、slash 菜单、审批反馈，交互流畅且可追溯。

Skills 发现

通过 `SKILL.md` 自动发现本地技能，让能力模块化、可复用、可共享。

MCP 工具接入

支持通过 stdio 动态加载 MCP 服务，并可读取 resources 与 prompts。

兼容 Anthropic 风格接口

保留通用配置方式，适应不同模型与环境，便于本地和团队统一部署。

工作流一眼看懂

MiniCode 的核心是一个清晰、可解释、可扩展的 agent loop。

1. 接收用户需求并解析目标
2. 检查工作区并按需调用工具
3. 输出变更 diff 并等待审批
4. 执行写入并在同一终端返回结果
